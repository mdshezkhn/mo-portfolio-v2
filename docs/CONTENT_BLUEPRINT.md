# Content Blueprint

**Version:** 1.1 (Approved)

## Rules
1. **Evidence Source mandatory** on every block: CV, employment record, certificate, PGCE work, quantitative data, or direct user statement. No source, no publish.
2. **Purpose mandatory** on every slot. A slot with no articulable purpose is deleted, even if content exists.
3. Placeholders: `[NEEDS INPUT: description]` – never shippable.
4. Voice per VISUAL_IDENTITY.md section 18.
5. Priority: **Must** = launch blocker · **Should** = credibility booster · **Nice** = v2.

## Slot Schema
| Slot | Limit | Priority | Evidence Source | Purpose |

## Section Slots (summary)
**Hero:** name/title (Must, user, position) · value proposition <=14 words (Must, user statement, position immediately) · portrait (Must, asset, humanize) · CV button + PDF (Must, asset, enable action) · location/availability (Should, user, answer logistics) · intro animation (Nice)

**Impact Highlights:** 3–4 stats, each = number + label + source context (Should, verified documents, prove scale/reach/improvement/leadership). Candidates pending verification: years in education, countries, years in China, schools/grades, GEDU training completion improvement, WhiteHat Jr quality work. **<3 verified → section ships hidden.**

**My Story:** narrative <=250 words (Must, user interview, emotional connection before evaluation) · values row (Should) · candid photo (Nice)

**Teaching Impact:** 4–6 cards, min 3 to launch (Must, per-card documented evidence, demonstrate measurable classroom value). Card: tag · headline <=10 words · 2–3 sentences · visible source line.

**Teaching Philosophy:** lead <=40 words + six topics 60–100 words each, total 450–600, cap 700 (Must, user beliefs + real examples, reveal how he thinks about learning)

**Leadership:** 2–4 cards (Must, evidence, demonstrate leadership beyond classroom) · 3–4 principles <=8 words (Should, user-authored, reveal decision-making philosophy)

**Professional Journey:** all roles with school/country/dates/title, major expanded, minor condensed (Must, CV/employment records, show trajectory)

**Research:** PGCE focus (Must) · inquiry approach (Should) · classroom-change statement (Must, connect research to practice) · publications slot (Nice). <=300 words total.

**Classroom Moments:** 8 min / 12–20 target images (Must) · caption + alt per image (Must, turn photos into stories) · categories (Should) · lightbox (Should). Safeguarding checklist gates every image.

**Credentials:** education + certifications with issuer/year (Must, certificates) · languages + skills (Must) · Currently Learning 2–4 genuine items (Should, signal growth) · safeguarding statement (Should)

**Contact:** closing line <=15 words · obfuscated email · Connect on LinkedIn + URL · Download CV (all Must, convert interest to action)

## Launch Gate
All Must slots filled with sourced approved content · Impact Highlights verified or hidden · every image consent-cleared · zero [NEEDS INPUT] markers in shipped sections.
