# videos/

Demo lesson recordings and professional video content.

## Naming Convention

```
demo-lesson-esl-grade5-2025.mp4
pd-workshop-gedu-2023.mp4
```

## Hosting Recommendation

For performance, **do not self-host videos** in this folder unless they are very short clips.

Instead:
- Upload to YouTube (unlisted) or Vimeo.
- Embed the video in `index.html` using an `<iframe>`.
- Use the `assets/videos/` folder only for short clips (under 5MB) or poster images.
